Sunshine & Co.

Sunshine & Co. is a website designed for a salon located in Saskatoon. The website offers a user-friendly interface for clients to explore a wide range of hair products and services offered by the salon. Users can browse through a catalog of hair care products, and view descriptions. Users can also view services offered by the employees of the salon. Additionally, the website features an appointment booking form.

Developed as a group project with students from Saskatchewan Polytechnic

Developed with:

HTML Bootstrap JavaScript

Pokemon Team Builder

PowerPoke is a comprehensive web application featuring an API and database functionality that enables users to interact with a vast collection of Pokémon. Users can create their own unique Pokémon and assemble them into custom teams. The application provides extensive information on each Pokémon, including stats, moves, abilities, and more. With PowerPoke, users can strategize and build teams tailored to their preferences.

Developed as a group project with students from Saskatchewan Polytechnic

Developed with:

HTML CSS Bootstrap Vue JavaScript TypeScript SQL

TRUE LiFESTiLE Design 3D Room Designer

The True Lifestyle 3D Room Designer is an innovative full-stack web application that appeals to interior designers. It offers both users and administrators robust functionalities to create and manage 3D room designs. Users can utilize the frontend interface to design virtual rooms and place virtual models of kitchen cabinets within them. The application provides a 3D rendering of the designed rooms, allowing users to visualize their ideas effectively. Administrators have access to a backend interface to manage database entries, insert new data, and oversee the application's operation.

Developed as a group project with students from Saskatchewan Polytechnic

Developed with:

HTML CSS Bootstrap React JavaScript Cypher
